研究所主修電腦圖學，論文研究如何利用 f-map 配合 Laplace Interpolation 將人體器官表面模型轉換成目標之體態。自學能力強，喜歡接觸新的知識、工具，對於 3D 建模有相當的了解。平常會住在 Blender StackExchange 裡面，得到超過 6000 的 Reputation，排名前一百名，心有戚戚焉。學習 Blender 的同時撰寫了許多 Addon，對於 Python 及 OOP 有深入的了解，如有需要 Addon，歡迎自取放在 Github。

人生至今有零星的撰寫網頁經驗，一路從 Frontpage、Dreamweaver 走到 wordpress 再到 bootstrap 最後投入 React 亞馬遜叢林生態系的懷抱，每天思考著我的人生是不是有人刻了一個模板出來，然後上帝 install 後就 build 出一個個的人，近期於中研院資創所投入開發 WWW 2020 conference 資訊網頁，使用 React 配合 Material UI 打造 Gatsby 產生的靜態網頁，並部屬 nginx server 於 AWS EC2 上，有完整的 SSL 自動憑證系統以及 Reverse Proxy。
